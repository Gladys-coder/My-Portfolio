How To Create The Image Of A Calculator

This Project was about how to Create the image of a calculator without JavaScript in it. We can do so by first writing the numbers and arranging them like how a calculator is. Then putting the div tag around the numbers and symbols. You then give a background color to all the numbers. You should put all the numbers in a section or a div and also set the padding of the section and a margin also. You can decide to set the background color of the div or section.
